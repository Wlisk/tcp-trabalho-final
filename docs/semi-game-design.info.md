# O QUE FAZEM?

O que certas entidades podem fazer no jogo (suas capacidades) e recursos inerentes ao jogo.

## O que o player pode fazer?

- atacar com ataques básicos
- atacar com ataques especiais
- defender contra ataques
- trocar de arma/armadura
- usar consumíveis (que dão buff/debuff)
- ganhar XP, money e outros ao vencer partida
- subir de nível ao coletar XP suficiente
- avançar 1 fase ao derrotar o chefão dela
- perder o jogo ao perder N vidas
- reiniciar a fase ao perder 1 vida
- perder HP ao ser atacado
- ganhar HP ao receber buff específico
- ataques tem chance de crítico

## O que os monstros podem fazer?

- atacar jogador
- se defender do jogador
- atacar o jogador com ataques especiais
- recuperar HP em cada turno
- perder HP ao ser atacado
- bloquear uma jogada do jogador (stun)
- ataques tem chance de crítico

## Como o usuário interage com o jogo?

- usando o teclado para ações no jogo (ataque, defesa, uso de consumíveis, etc.)
  - tecla A ataca
  - tecla D defende
  - teclas 1, 2, 3 usam ataques especiais respectivos

- usando o mouse para ações no jogo (mesmo que o do teclado, mas no "modo gráfico")
  - clica nas ações
  - move elementos para reposicionalos

## Como é o game design do jogo?

A história de um herói que entra numa dungeon infestada de monstros para derrotálos e salvar uma cidade de sua possível destruição.

- há 3 classes (swordsman, mage e archer) para serem escolhidas
- o jogo é de turnos, após uma ação do jogador, a próxima é do monstro e vice-versa
- há X fases no total, com 1 chefão cada (um mais forte que o outro)
- o jogador tem N vidas, se perder todas, isto é , as vidas chegarem a zero, perde o jogo e o jogo acaba
- quando o jogador derrota um chefão da fase, ele avança para a próxima fase
- quando o jogador derrota o último chefão da última fase ele vence o jogo e o jogo acaba
- se o jogo acabar, de qualquer maneira que seja, ele volta para o menu inicial
- os monstros/chefões "derrubam" (drop) itens no jogo (armas/armaduras ou consumíveis) que podem ser usados pelo jogador na próxima fase
